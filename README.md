## About the Game
### plants
#### Sunflower (summons sunlight, click for 5 gold)
#### Peashooter (shoots pea bullet at zombies)
#### Chickenflower (summons chicken, can click for 100 gold)
#### Basketball-shooter (shoots basketballs at zombies)

### zombies
#### Normal zombies
#### Kunkun zombies
#### Mamba zombies



To play:
```bash
python3 -m ./src/main
```
